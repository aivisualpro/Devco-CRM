# Cache Tags

This document lists all the cache tags introduced for `unstable_cache` optimization.

| Tag | Route | Description |
|---|---|---|
| `schedule-stats` | `/api/schedules` (`getScheduleStats`) | Monthly payroll totals and timesheet aggregations. |
| `schedule-counts` | `/api/webhook/devcoBackend` (`getEstimateStats`) | Schedule counts and estimate counts aggregation. |
| `wip-calculations` | `/api/quickbooks/projects` (GET) | WIP calculations for project profitability. |
